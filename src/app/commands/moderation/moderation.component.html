Moderation

Command Descriptor Example Usage
c!prune (amount) Clears all messages within the given number c!prune 50
c!announce (channel) (announcement) Makes an announcement to the chosen channel with the announcement c!announce #announcements I'm the best because I use Campfire!
c!embed (message) Makes an embed with the message c!embed woah cool
c!say (message) Says the message c!say coool
c!ban (user) (reason) Bans a user with a given reason c!ban @Marshmellow#4434 Spamming
c!kick (user) (reason) Kicks a user with a given reason c!kick @Marshmellow#4434 Abusing caps
